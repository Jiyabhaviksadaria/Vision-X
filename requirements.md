# Project Requirements
team name :- Vision-X
## 1. Project Title
AI-Powered Purchase Decision Support System  
(AI for Bharat Hackathon)

## 2. Problem Statement
Consumers—especially in Bharat (Tier 2/3 cities and rural areas)—often struggle to make informed purchase decisions due to:
- Information overload
- Fake or biased reviews
- Language barriers
- Lack of personalized recommendations

This project aims to solve these issues using AI-driven insights delivered in a simple, multilingual, and user-friendly manner.

## 3. Objectives
- Help users make smarter purchase decisions
- Provide summarized, trustworthy product insights
- Support regional Indian languages
- Enable comparison and recommendation using AI

## 4. Target Users
- Online shoppers in India
- First-time digital buyers
- Users from Tier 2, Tier 3, and rural regions
- Price-sensitive consumers

## 5. Functional Requirements
### 5.1 User Features
- User can search for a product by name or category
- User can compare multiple products
- User receives AI-generated product summaries
- User sees pros & cons extracted from reviews
- User gets recommendations based on preferences
- User can select preferred language (Hindi, English, etc.)

### 5.2 System Features
- Collect product data (price, ratings, reviews)
- Perform sentiment analysis on reviews
- Generate concise summaries using LLM
- Rank products based on value-for-money
- Store user preferences for personalization

## 6. Non-Functional Requirements
- Fast response time (< 3 seconds)
- Scalable architecture
- Secure handling of user data
- High availability
- Easy-to-use UI for low digital literacy users

## 7. Assumptions
- Internet access is available
- Product data APIs or datasets are accessible
- Users have basic smartphone familiarity

## 8. Constraints
- Limited hackathon development time
- API usage limits
- Budget constraints for cloud services

## 9. Success Metrics
- Reduction in user decision time
- User satisfaction score
- Accuracy of recommendations
- Engagement rate
